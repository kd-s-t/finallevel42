# Theme Colors

## Color Palette

### Primary - Midnight Blue
**Hex:** `#0A1A2F`

**Usage:**
- Background
- Main panels
- Navigation bars

---

### Accent - Electric Cyan
**Hex:** `#00E5FF`

**Usage:**
- Highlights
- "42" branding
- XP bars
- Buttons

---

### Secondary Gray - Cool Gray
**Hex:** `#8FA3AD`

**Usage:**
- Text
- Subtext
- Charts

---

### Text / White
**Hex:** `#FFFFFF`

**Usage:**
- Main text on dark background

---

## CSS Variables Reference

For implementation, consider using these CSS variables:

```css
:root {
  --color-primary: #0A1A2F;
  --color-accent: #00E5FF;
  --color-secondary-gray: #8FA3AD;
  --color-text-white: #FFFFFF;
}
```
